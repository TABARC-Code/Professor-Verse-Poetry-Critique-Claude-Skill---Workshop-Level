## One-Line Description

Workshop-level poetry critique — diagnose what works, what fails, why it matters, and how to fix it.

---

## What It Does

Critiques poems at any level (beginner to MFA). Handles form, prosody, revision, manuscript work, teaching, exam prep (AP and UK), code tools, and hybrid poetry-code work. Operates in nine modes. Includes POE-01 to POE-15 command codes for routing common tasks.

---

## Core Capabilities

- **Critique modes:** Feedback, Brutal Professor, Line Editor, Formalist, Contemporary Poetry Editor, Revision Architect, Teaching, Diagnostic, MFA Workshop
- **Forms:** Sonnet, villanelle, sestina, pantoum, ghazal, haiku, tanka, sapphics, ode, ballad, dramatic monologue, prose poem, and more
- **Diagnostics:** Prosody, scansion, metre, rhyme, lineation, diction, syntax, imagery, tone, voice, closure, structure, form satisfaction
- **Output types:** Revision plans, manuscript audits, grade notes, study materials, teaching packets, annotation templates, schemas, rubrics
- **Exam support:** AP poetry (thesis + evidence + commentary); UK (GCSE, A-level, unseen, PEE/PEEL, AQA/OCR)
- **Teaching:** Craft concepts at beginner, high school, undergraduate, MFA, and general reader level
- **Code tools:** Manuscript analyser (word frequency, abstraction detection), draft comparison, submission trackers, hybrid poetry-code tools
- **Spoken word:** Cadence, anaphora, performance technique, rap prosody, dub, slam
- **Traditions:** Free verse, formalism, Language poetry, confessional, workshop, experimental, postcolonial, ecopoetry, trauma poetry

---

## What Makes It Different

- **Not generic.** Diagnosis is specific to the poem's tradition. Free verse gets different standards than a ghazal.
- **Not flattering.** Feedback is direct, evidence-based, craft-focused. No padding.
- **Not vague.** "Add more imagery" → specific lineation and word-choice edits.
- **Workshop-grounded.** Built on actual critique practice, not writing advice templates.
- **Precise tradition lens.** Confessional voice + Language poetry syntax = reads both traditions, identifies productive vs. confused tension.
- **Quality-assured.** Built-in contradiction checks, failure mode detection, peer review before responding.

## Quick Start

```
Critique this poem. Tell me what's working, what's failing, and why.

[paste poem]
```

For modes:
- "Line by line workshop" → Line Editor Mode
- "Be brutal" → Brutal Professor Mode
- "Is this publishable?" → Contemporary Poetry Editor Mode
- "How do I fix this?" → Revision Architect Mode
- "Teach me why this works" → Teaching Mode

For command codes:
```
Use POE-03 and POE-05: Analyse theme, imagery, and what each device does.
```

For exams:
- AP: route to study-assignments-exams-and-classroom-tools.md
- UK (GCSE/A-level): route to uk-exam-support.md
